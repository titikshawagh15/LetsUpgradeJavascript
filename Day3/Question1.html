Explore and explain the various methods in console function:

1. console.log(): Used to log/print the output to the console. We can put any type inside the log() ie. string, array,object, etc. Example:

Eg.-console.log('abc')
console.log(true)

2. console.error(): Used to log error message to console. Used in testing of code. Uses red color.

Eg.-console.error('This is console error')

3. console.warn(): Used to give warning message to console. Uses yellow color.

Eg.-console.warn('This is warning')

4. console.count(): Used to count number that the function hit by this counting method.

Eg.-for(let i=0;i<4;i++)
{
console.count(i);
}

5. console.group() and console.groupEnd(): Used to group contents in a separate block, which will be indented.

Eg.-console.group('simple')
console.warn('WARNING')
console.error('ERROR')
console.groupEnd('simple')

Write a brief intro on available datatypes in javascript

Number:
The number type represents both integer and floating point numbers. Operations like multiplication,division,addition,subtraction,etc can be performed on numbers.
Eg- let n=123;
n=12.345;

BigInt:
BigInt type was recently added to the language to represent integers of arbitrary lenth. A bigInt value is created by appending 'n' to end of an integer.
Eg- const bigInt=124262423273264263551172n

String:
String is used for collection of characters. They must be surrounded by quotes. Double quotes("Hello") and single quotes('Hello') are simple quotes. Backticks(`Hello`) are used for extended functionality.
Eg- let str="ABC"

Boolean:
The Boolean type has only two values 'true' and 'false'. It stores yes/no values. Boolean values also comes as a result of comparisons.
Eg- let a=true let b=false

Null:
This type contains only 'null' value. It represents "nothing","empty" or "value unknown".
Eg- let age = null
